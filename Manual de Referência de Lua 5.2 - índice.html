Manual de Referência de Lua 5.2

Essa é uma tradução para o português do manual de referência de Lua, que é a definição oficial da linguagem Lua.
Para uma introdução completa à programação em Lua, veja o livro Programming in Lua.

A tradução para o português foi feita por Sérgio Queiroz de Medeiros com revisão de Roberto Ierusalimschy.
